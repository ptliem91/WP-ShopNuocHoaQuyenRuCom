The plugin could help to hide "Add to Cart" button for one or more Woocommerce products. 

== Description ==
It will :

- Hide Add to Cart button from product single page.
- Hide Add to Cart button from category page.
- Hide Add to Cart button from homepage and all other pages.

You can set the "expiry date" and the "Add to cart" button will come back afer that date.

== Installation ==
1. Upload `hide-add-to-cart-button` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
